BSBL - The Block Sparse Bayesian Learning Framework

I developing fast implementations of the Block Sparse Bayesian Learning (BSBL, see Zhilin Zhang's Website) Framework, using the Fast Marginalized Likelihood Maximization method (see Fast marginal likelihood maximisation for sparse Beysian models for some basics on this method).

BSBL-FM - A Fast Marginalized Implementation

Basic

The BSBL-FM is a fast marginalized implementation of the BSBL framework. It is based on the BCS (Ji2008) and fast RVM (Tipping2003). The major work are the block extension and the capability of exploiting the intra-block correlation. It is extremely fast when the signal has the block structure and the normlized sparsity level is not high.
